# 📊 ACF vs PACF in Time Series Analysis

---

## 📌 Introduction

In **time series analysis**, understanding how past values influence current values is very important.

Two powerful tools used for this are:

* **ACF (Autocorrelation Function)**
* **PACF (Partial Autocorrelation Function)**

These help us:

* Understand patterns in data
* Identify model type (AR, MA, ARIMA)

---

# 🔹 1. What is ACF (Autocorrelation Function)?

## ✅ Definition

ACF measures the **correlation between a time series and its past values (lags)**.

👉 Simple Meaning:

> “How much does today depend on previous days?”

---

## 📐 Formula (Easy View)

```
ACF(k) = Corr(X_t , X_{t-k})
```

Where:

* `k` = lag (how many steps back)
* `X_t` = current value
* `X_{t-k}` = past value

---

## 📊 Interpretation

* High value at lag 1 → strong relation with previous value
* Slowly decreasing values → AR-type behavior
* Sudden cutoff → MA-type behavior

---

## 🧠 Example

If today's sales depend on yesterday’s sales, then:

* ACF at lag 1 will be high

---

# 🔹 2. What is PACF (Partial Autocorrelation Function)?

## ✅ Definition

PACF measures the **direct relationship between current value and a specific lag**, removing effects of intermediate lags.

👉 Simple Meaning:

> “How much does today depend ONLY on lag k (ignoring others)?”

---

## 🧠 Key Idea

* ACF = Direct + Indirect effects
* PACF = Only Direct effects

---

## 📊 Interpretation

* Sharp cutoff after lag p → AR(p)
* Gradual decay → MA model

---

## 🧠 Example

If:

* Day 3 affects Day 4
* Day 2 affects Day 3

Then:

* ACF includes both effects
* PACF removes indirect effect of Day 2 via Day 3

---

# 🔹 3. ACF vs PACF (Comparison Table)

| Feature          | ACF               | PACF               |
| ---------------- | ----------------- | ------------------ |
| Measures         | Total correlation | Direct correlation |
| Indirect effects | Included          | Removed            |
| Used for         | MA(q) model       | AR(p) model        |
| Pattern          | Tails off         | Cuts off           |

---

# 🔹 4. Model Identification Using ACF & PACF

## ✅ AR Model (p)

* PACF → **cuts off after lag p**
* ACF → gradual decay

---

## ✅ MA Model (q)

* ACF → **cuts off after lag q**
* PACF → gradual decay

---

## ✅ ARMA Model

* Both ACF & PACF → gradual decay

---

# 🔹 5. Simple Real Example

### 📊 Dataset (Daily Temperature)

| Day | Temperature |
| --- | ----------- |
| 1   | 30          |
| 2   | 32          |
| 3   | 31          |
| 4   | 33          |

---

### 🔍 ACF Checks

* Day 4 vs Day 3 → Lag 1
* Day 4 vs Day 2 → Lag 2

---

### 🔍 PACF Checks

* Direct effect of Day 2 on Day 4
* Removes effect of Day 3

---

# 🔹 6. Important Points

* Values range from **-1 to +1**
* Significant spikes → meaningful relationships
* Used after making data **stationary**

---

# 🔹 7. Python Implementation

```python
import pandas as pd
import matplotlib.pyplot as plt
from statsmodels.graphics.tsaplots import plot_acf, plot_pacf

# Example data
data = [30, 32, 31, 33, 35, 36, 38]
ts = pd.Series(data)

# ACF plot
plot_acf(ts)
plt.title("ACF Plot")
plt.show()

# PACF plot
plot_pacf(ts)
plt.title("PACF Plot")
plt.show()
```

---

# 🔹 8. When to Use What (Quick Guide)

| Situation               | Use  |
| ----------------------- | ---- |
| Want to detect MA order | ACF  |
| Want to detect AR order | PACF |
| Building ARIMA model    | Both |

---

# 🔹 9. Memory Trick 🧠

```
ACF → MA model
PACF → AR model
```

---

# 🚀 Final Summary

* **ACF** tells overall dependency on past values
* **PACF** tells direct dependency only
* Both are essential for **time series modeling**
* Help in choosing correct model (AR, MA, ARIMA)
