## 1️⃣ Introduction

Time series forecasting is used to predict future values based on past observations.

One of the most fundamental models in time series is the **AutoRegressive (AR) model**.

---

## 2️⃣ What is AutoRegressive (AR)?

**AutoRegressive (AR)** means:

> The current value of a series depends on its **past values**.

### 🔹 Simple Idea

* Today’s value depends on yesterday’s value
* Past influences the present

---

## 3️⃣ AR(1) Mathematical Model

The simplest AR model is AR(1):

Yₜ = c + φYₜ₋₁ + εₜ

### 🔹 Where:

| Symbol | Meaning                      |
| ------ | ---------------------------- |
| Yₜ     | Current value                |
| Yₜ₋₁   | Previous value               |
| c      | Constant term                |
| φ      | Coefficient (impact of past) |
| εₜ     | Random error (noise)         |

---

## 4️⃣ Example Dataset

Let’s consider a small time series dataset:

| Time (t) | Value (Yₜ) |
| -------- | ---------- |
| 1        | 10         |
| 2        | 12         |
| 3        | 13         |
| 4        | 15         |
| 5        | 18         |

---

## 5️⃣ Step-by-Step AR Calculation

### 🔹 Assumptions

* c = 2
* φ = 0.8

### 🔹 Model Used

Yₜ = 2 + 0.8Yₜ₋₁

---

### 🔹 Calculation Table

| Time (t) | Actual Yₜ | Previous Yₜ₋₁ | Predicted Yₜ = 2 + 0.8Yₜ₋₁ |
| -------- | --------- | ------------- | -------------------------- |
| 1        | 10        | —             | —                          |
| 2        | 12        | 10            | 10                         |
| 3        | 13        | 12            | 11.6                       |
| 4        | 15        | 13            | 12.4                       |
| 5        | 18        | 15            | 14                         |

---

### 🔹 Detailed Calculations

* For t = 2 → 2 + 0.8 × 10 = 10
* For t = 3 → 2 + 0.8 × 12 = 11.6
* For t = 4 → 2 + 0.8 × 13 = 12.4
* For t = 5 → 2 + 0.8 × 15 = 14

---

## 6️⃣ Interpretation of Results

* The model uses **previous values to predict current values**
* Predictions are smoother than actual values
* The model lags slightly behind real data

---

## 7️⃣ Key Intuition

👉 If yesterday’s value is high → today’s value will likely be high

👉 If yesterday’s value is low → today’s value will likely be low

### 🔹 Important Insight

* AR models rely only on **past values of the same series**
* No external variables are used

---

## 8️⃣ Real-World Applications

AR models are widely used in:

* 📈 Stock price prediction
* 🌡 Temperature forecasting
* 🛒 Sales forecasting
* 📊 Economic data analysis

---

## 9️⃣ Summary

| Concept  | Description                               |
| -------- | ----------------------------------------- |
| AR Model | Uses past values to predict current value |
| AR(1)    | Uses 1 previous value                     |
| Formula  | Yₜ = c + φYₜ₋₁                            |
| Nature   | Depends only on historical data           |

---

## 🚀 Final Takeaway

> **AutoRegressive models are the foundation of time series forecasting.**

They are simple, intuitive, and powerful for modeling temporal dependencies.
