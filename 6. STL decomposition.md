# Time Series Decomposition: Classical vs STL

---

## 1. What is Decomposition?

Think of a time series like this:

👉 **Your data = Trend + Seasonality + Noise**

### Example:

* Sales increasing every year → **Trend**
* Sales spike every December → **Seasonality**
* Random ups & downs → **Noise**

---

## 2. Classical Decomposition (OLD way)

### How it thinks:

> “Seasonality is fixed and never changes”

### How it works:

* Uses simple averages (**moving average**)
* Finds:

  * **Trend** → by smoothing
  * **Seasonality** → same pattern repeated every cycle

### Example:

Imagine:

* Every year December sales = +100 always
* Every January = -50 always

👉 Classical says:

> “Perfect, this pattern is constant forever”

### Problem:

Real life is **NOT like this**.

* December sales in 2020 = 100
* December sales in 2025 = 300

👉 Pattern changed ❌
👉 Classical fails here

---

## 3. STL Decomposition (MODERN way)

STL uses:

👉 **LOESS (a smart smoothing technique)**

### How it thinks:

> “Let me look at small chunks of data and adapt”

### How it works:

* Looks **locally (nearby data points)**
* Adjusts **trend and seasonality over time**

### Example:

December sales:

* 2020 → 100
* 2021 → 150
* 2022 → 220

👉 STL says:

> “Seasonality is increasing — I’ll adjust it”

✔️ Learns **changing patterns**

---

## 4. Core Difference (SUPER SIMPLE)

| Feature        | Classical Decomposition  | STL Decomposition |
| -------------- | ------------------------ | ----------------- |
| Seasonality    | Fixed                    | Changes over time |
| Approach       | Global                   | Local (adaptive)  |
| Method         | Moving Average           | LOESS smoothing   |
| Flexibility    | Low                      | High              |
| Real-world fit | Poor (for changing data) | Excellent         |

---

## 5. Real-life Analogy

* **Classical = Old teacher**

  * Teaches same notes every year
  * Doesn’t update

* **STL = Smart teacher**

  * Updates notes every year
  * Adjusts based on new trends

---

## 6. One-line Difference

👉 **Classical = rigid (fixed pattern)**
👉 **STL = flexible (changing pattern)**
