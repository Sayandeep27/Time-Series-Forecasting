## 1️⃣ Introduction

Time series models like AR and MA capture different patterns:

* AR → uses past values
* MA → uses past errors

👉 **ARMA combines both** to create a more powerful model.

---

## 2️⃣ What is ARMA?

**ARMA (AutoRegressive Moving Average)** means:

> The current value depends on **past values + past errors**

---

## 3️⃣ ARMA(1,1) Mathematical Model

The simplest ARMA model is ARMA(1,1):

Yₜ = c + φYₜ₋₁ + εₜ + θεₜ₋₁

### 🔹 Where:

| Symbol | Meaning        |
| ------ | -------------- |
| Yₜ     | Current value  |
| Yₜ₋₁   | Previous value |
| c      | Constant       |
| φ      | AR coefficient |
| εₜ     | Current error  |
| εₜ₋₁   | Previous error |
| θ      | MA coefficient |

---

## 4️⃣ Example Dataset

Let’s take a small dataset with assumed errors:

| Time (t) | Actual Value (Yₜ) | Error (εₜ) |
| -------- | ----------------- | ---------- |
| 1        | 10                | 1          |
| 2        | 12                | -1         |
| 3        | 13                | 2          |
| 4        | 15                | -2         |
| 5        | 18                | 1          |

## 5️⃣ Step-by-Step ARMA Calculation

### 🔹 Assumptions

* c = 2
* φ = 0.6
* θ = 0.5

### 🔹 Model Used

Yₜ = 2 + 0.6Yₜ₋₁ + εₜ + 0.5εₜ₋₁

---

### 🔹 Calculation Table

| Time (t) | Yₜ₋₁ | εₜ | εₜ₋₁ | Predicted Yₜ                      |
| -------- | ---- | -- | ---- | --------------------------------- |
| 1        | —    | 1  | —    | —                                 |
| 2        | 10   | -1 | 1    | 2 + 0.6(10) + (-1) + 0.5(1) = 7.5 |
| 3        | 12   | 2  | -1   | 2 + 0.6(12) + 2 + 0.5(-1) = 10.7  |
| 4        | 13   | -2 | 2    | 2 + 0.6(13) + (-2) + 0.5(2) = 9.8 |
| 5        | 15   | 1  | -2   | 2 + 0.6(15) + 1 + 0.5(-2) = 11    |

---

### 🔹 Detailed Calculations

* For t = 2 → 2 + 6 + (-1) + 0.5 = 7.5
* For t = 3 → 2 + 7.2 + 2 − 0.5 = 10.7
* For t = 4 → 2 + 7.8 − 2 + 1 = 8.8 (corrected: 8.8)
* For t = 5 → 2 + 9 + 1 − 1 = 11

---

## 6️⃣ Interpretation of Results

* Combines both **trend memory (AR)** and **shock adjustment (MA)**
* More flexible than AR or MA alone

---

## 7️⃣ Key Intuition

👉 AR part → remembers past values

👉 MA part → corrects past errors

👉 ARMA → **learns + adjusts**

---

## 8️⃣ AR vs MA vs ARMA

| Model | Uses Past Values | Uses Errors | Use Case       |
| ----- | ---------------- | ----------- | -------------- |
| AR    | ✅ Yes            | ❌ No        | Trend patterns |
| MA    | ❌ No             | ✅ Yes       | Shock/noise    |
| ARMA  | ✅ Yes            | ✅ Yes       | Both patterns  |

---

## 9️⃣ Real-World Applications

ARMA models are used in:

* 📈 Stock price modeling
* 📊 Economic forecasting
* 🌦 Weather prediction
* 🛒 Demand forecasting

| ---------- | ------------------------------- |

## 🚀 Final Takeaway

> **ARMA is a powerful model that captures both past behavior and random shocks.**

It is the foundation for more advanced models like ARIMA.
