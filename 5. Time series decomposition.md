## 📖 Introduction

**Time Series Analysis** is a statistical technique used to analyze data points collected over time. One of the most important concepts in time series is **decomposition**, which helps break down complex data into understandable components.

---

## 🔍 What is Time Series Decomposition?

**Time Series Decomposition** is the process of splitting a time series into multiple underlying components to better understand patterns and structure.

### 👉 Objective:

* Understand underlying patterns
* Identify seasonality and trends
* Improve forecasting accuracy

---

## 🧩 Components of Time Series

A time series typically consists of the following four components:

### 1. Trend (T)

* Long-term movement of data
* Indicates overall direction (upward/downward)
* Example: Increasing yearly sales

### 2. Seasonality (S)

* Repeating patterns at fixed intervals
* Example: Higher sales during festivals

### 3. Cyclicality (C)

* Long-term fluctuations without fixed period
* Example: Economic cycles (boom & recession)

### 4. Irregular/Residual (I or R)

* Random noise
* Unpredictable variations

---

## ➕ Additive Model

### 📌 Formula

```math
Y = T + S + C + I
```

### 📖 Explanation

* Components are **added together**
* Each component contributes independently

### ✅ When to Use

* Seasonal variation is **constant**
* Fluctuations remain similar over time

### 📊 Example

| Time | Trend | Seasonality | Final Value |
| ---- | ----- | ----------- | ----------- |
| Jan  | 100   | +10         | 110         |
| Feb  | 100   | -5          | 95          |

### 💡 Key Insight

* Seasonal effect does **not depend on trend**

---

## ✖️ Multiplicative Model

### 📌 Formula

```math
Y = T \times S \times C \times I
```

### 📖 Explanation

* Components are **multiplied**
* Seasonal effect varies with trend

### ✅ When to Use

* Seasonal variation **changes with level**
* Data shows increasing/decreasing fluctuations

### 📊 Example

| Time | Trend | Seasonality | Final Value |
| ---- | ----- | ----------- | ----------- |
| Jan  | 100   | 1.10        | 110         |
| Feb  | 200   | 1.10        | 220         |

### 💡 Key Insight

* Seasonal effect is **proportional to trend**

---

## ⚖️ Additive vs Multiplicative

| Feature         | Additive Model    | Multiplicative Model   |
| --------------- | ----------------- | ---------------------- |
| Formula         | Y = T + S + C + I | Y = T × S × C × I      |
| Seasonal Effect | Constant          | Proportional           |
| Variation       | Same size         | Changes with level     |
| Use Case        | Stable data       | Growing/shrinking data |

---

## ⚙️ How Decomposition Works

### Step-by-Step Process

1. Estimate **Trend** (moving average / smoothing)
2. Remove trend from series
3. Extract **Seasonality**
4. Remaining = **Residual (Noise)**

## 🐍 Python Implementation

### 📦 Install Required Libraries

```bash
pip install statsmodels matplotlib pandas
```

### 📊 Code Example

```python
import pandas as pd
import matplotlib.pyplot as plt
from statsmodels.tsa.seasonal import seasonal_decompose

# Example time series data
data = pd.read_csv("data.csv", parse_dates=["Date"], index_col="Date")

# Perform decomposition
result_add = seasonal_decompose(data, model='additive')
result_mul = seasonal_decompose(data, model='multiplicative')

# Plot results
result_add.plot()
plt.show()

result_mul.plot()
plt.show()
```

---

## 🎯 When to Use Which Model

### Use Additive Model if:

* Data has constant variance
* Seasonal fluctuations are stable

### Use Multiplicative Model if:

* Variance increases over time
* Seasonal effect grows with trend

---

## 🧠 Summary

* Time series = combination of **Trend + Seasonality + Cyclicality + Noise**
* **Additive model** assumes constant seasonal variation
* **Multiplicative model** assumes proportional variation
* Choosing the correct model improves forecasting accuracy
