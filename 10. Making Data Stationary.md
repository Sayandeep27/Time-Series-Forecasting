# 📊 Making Time Series Data Stationary

---

## 📌 What is Stationarity?

A **stationary time series** is one whose statistical properties remain constant over time.

### A time series is stationary if:

* Mean is constant
* Variance is constant
* Covariance depends only on lag (not time)

---

## ❗ Why Do We Need Stationarity?

Most time series models like:

* ARIMA
* SARIMA
* Statistical forecasting models

👉 Assume data is stationary.

If data is **non-stationary**, models may:

* Give poor predictions
* Capture false patterns
* Become unreliable

---

# 🔧 Methods to Make Data Stationary

---

# 1️⃣ Differencing (Removing Trend)

---

## 📌 Definition

Differencing means subtracting the previous value from the current value.

```math
Y'_t = Y_t - Y_{t-1}
```

---

## 💡 Intuition

Instead of looking at actual values, we focus on **changes between values**.

---

## 📊 Example

### Original Data

| Time | Value |
| ---- | ----- |
| 1    | 100   |
| 2    | 120   |
| 3    | 140   |

### After Differencing

| Time | Value |
| ---- | ----- |
| 2    | 20    |
| 3    | 20    |

👉 Trend is removed → Data becomes stable

---

## 🧠 When to Use?

* Data has **upward or downward trend**

---

## 🔁 Types of Differencing

### 1. First Order Differencing

```math
Y'_t = Y_t - Y_{t-1}
```

---

### 2. Second Order Differencing

```math
Y''_t = Y'_t - Y'_{t-1}
```

👉 Used when first differencing is not enough

---

# 2️⃣ Transformation (Fixing Variance)

---

## 📌 Definition

Transformation changes the scale of data to stabilize variance.

---

## ❗ Problem

Sometimes data shows:

* Small fluctuations initially
* Large fluctuations later

👉 Variance is not constant

---

## 💡 Intuition

Compress large values so variation becomes stable.

---

## 🔧 Common Transformations

### 1. Log Transformation

```math
Y' = log(Y)
```

---

### 2. Square Root Transformation

```math
Y' = √Y
```

---

### 3. Box-Cox Transformation

```math
Y' = (Y^λ - 1)/λ
```

---

## 🧠 When to Use?

* Variance increases over time
* Data shows exponential growth

---

# 3️⃣ Detrending (Removing Trend Directly)

---

## 📌 Definition

Detrending removes the trend component explicitly from the data.

---

## 💡 Intuition

Remove the overall direction and keep only fluctuations.

---

## 🔧 Methods

### 1. Linear Regression

Fit a line:

```math
Y_t = a + bt
```

Then subtract:

```math
Detrended = Original - Trend
```

---

### 2. Moving Average

Smooth data and subtract it.

---

## 🧠 When to Use?

* Trend is smooth and predictable

---

## ⚖️ Differencing vs Detrending

| Feature    | Differencing            | Detrending           |
| ---------- | ----------------------- | -------------------- |
| Approach   | Subtract previous value | Fit and remove trend |
| Complexity | Simple                  | Moderate             |
| Accuracy   | Approximate             | More precise         |

---

# 4️⃣ Seasonality Adjustment (Removing Repeating Patterns)

---

## 📌 Definition

Seasonality adjustment removes repeating patterns at fixed intervals.

---

## 💡 Example

Ice cream sales:

* High in summer
* Low in winter

👉 Pattern repeats every year

---

## 🔧 Methods

### 1. Seasonal Differencing

```math
Y'_t = Y_t - Y_{t-s}
```

Where:

* `s` = season length (e.g., 12 for monthly data)

---

### 2. Decomposition

Break data into:

```text
Time Series = Trend + Seasonality + Noise
```

Then remove seasonality.

---

## 🧠 When to Use?

* Data has repeating cycles

---

# 🧾 Final Summary Table

---

| Problem in Data   | Solution           | Method                    |
| ----------------- | ------------------ | ------------------------- |
| Trend             | Remove trend       | Differencing / Detrending |
| Changing Variance | Stabilize variance | Transformation            |
| Seasonality       | Remove cycles      | Seasonal Adjustment       |

---

# 🧠 Master Intuition

---

👉 Your data is made of:

```text
Time Series = Trend + Seasonality + Noise
```

---

👉 To make it stationary:

* Remove **Trend** → Differencing / Detrending
* Remove **Seasonality** → Seasonal Adjustment
* Fix **Variance** → Transformation

---

👉 Final result:

```text
Stationary Data = Only Noise (Stable Behavior)
```

## 📊 Load Data

```python
import pandas as pd
import matplotlib.pyplot as plt

# Example dataset
# Replace with your dataset

df = pd.read_csv("data.csv", parse_dates=['Date'], index_col='Date')

series = df['Value']
series.plot(title="Original Data")
plt.show()
```

---

## 1️⃣ Differencing

```python
diff = series.diff().dropna()
diff.plot(title="After Differencing")
plt.show()
```

---

## 2️⃣ Log Transformation

```python
import numpy as np

log_series = np.log(series)
log_series.plot(title="Log Transformed")
plt.show()
```

---

## 3️⃣ Seasonal Differencing

```python
seasonal_diff = series.diff(12).dropna()
seasonal_diff.plot(title="Seasonal Differencing")
plt.show()
```

---

## 4️⃣ Detrending (Rolling Mean)

```python
trend = series.rolling(window=12).mean()
detrended = series - trend

detrended.dropna().plot(title="Detrended Data")
plt.show()
```

---

## 📉 Stationarity Test (ADF Test)

```python
from statsmodels.tsa.stattools import adfuller

result = adfuller(series.dropna())

print("ADF Statistic:", result[0])
print("p-value:", result[1])
```

---

## 📌 Interpretation

| p-value | Meaning                |
| ------- | ---------------------- |
| < 0.05  | Data is stationary     |
| > 0.05  | Data is not stationary |
