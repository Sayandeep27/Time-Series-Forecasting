# 🔷 Introduction

Time Series Forecasting is used to predict future values based on past data. One of the most powerful and widely used statistical models for this purpose is **ARIMA**.

---

# 🔷 What is ARIMA?

**ARIMA = AutoRegressive Integrated Moving Average**

It is a combination of three models:

| Component | Full Form      | Purpose               |
| --------- | -------------- | --------------------- |
| AR (p)    | AutoRegressive | Uses past values      |
| I (d)     | Integrated     | Makes data stationary |
| MA (q)    | Moving Average | Uses past errors      |

👉 General Form:

```
ARIMA(p, d, q)
```

---

# 🔷 Components of ARIMA

---

## 🔶 1. AutoRegressive (AR - p)

Uses past values to predict the current value.

### Formula:

```
\[
Y_t = c + \phi_1 Y_{t-1} + \phi_2 Y_{t-2} + \cdots + \epsilon_t
\]
```

---

## 🔶 2. Integrated (I - d)

Used to make time series **stationary**.

### Differencing:

```
\[
Y'_t = Y_t - Y_{t-1}
\]
```

---

## 🔶 3. Moving Average (MA - q)

Uses past forecast errors.

### Formula:

```
\[
Y_t = \mu + \epsilon_t + \theta_1 \epsilon_{t-1}
\]
```

# 🔷 Example Dataset

Monthly sales data:

| Month | Sales |
| ----- | ----- |
| Jan   | 100   |
| Feb   | 110   |
| Mar   | 115   |
| Apr   | 130   |
| May   | 145   |
| Jun   | 150   |

👉 Observation:

* Upward trend present
* Data is NOT stationary

---

# 🔷 Step-by-Step Explanation

---

## Step 1: Check Stationarity

* Data has trend → Not stationary

---

## Step 2: Apply Differencing

```
\[
Y'_t = Y_t - Y_{t-1}
\]
```

| Month | Sales | Difference |
| ----- | ----- | ---------- |
| Feb   | 110   | 10         |
| Mar   | 115   | 5          |
| Apr   | 130   | 15         |
| May   | 145   | 15         |
| Jun   | 150   | 5          |

👉 Now data is approximately stationary

So, **d = 1**

---

## Step 3: Choose p and q

| Tool | Purpose           |
| ---- | ----------------- |
| PACF | Find p (AR order) |
| ACF  | Find q (MA order) |

---

# 🔷 Mathematical Formulation

For ARIMA(1,1,1):

```
\[
Y'_t = c + \phi_1 Y'_{t-1} + \theta_1 \epsilon_{t-1} + \epsilon_t
\]
```

---

# 🔷 Forecasting Example

Given:

* Last value = 150
* Last difference = 5
* AR coefficient (φ1) = 0.7
* MA coefficient (θ1) = 0.5
* Last error = 4

### Step 1: Predict differenced value

```
\[
Y'_t = 0.7 \times 5 + 0.5 \times 4
\]
\[
Y'_t = 3.5 + 2 = 5.5
\]
```

### Step 2: Convert back to original scale

```
\[
Y_t = 150 + 5.5 = 155.5
\]
```

👉 Forecast ≈ **156**

---

# 🔷 Model Building Steps

1. Check stationarity (ADF Test)
2. Apply differencing → find d
3. Plot ACF → find q
4. Plot PACF → find p
5. Train ARIMA model
6. Validate model
7. Forecast future values

---

# 🔷 Python Implementation

```python
import pandas as pd
from statsmodels.tsa.arima.model import ARIMA

# Sample Data
data = [100,110,115,130,145,150]
series = pd.Series(data)

# Build Model
model = ARIMA(series, order=(1,1,1))
model_fit = model.fit()

# Forecast
forecast = model_fit.forecast(steps=1)
print(forecast)
```

---

# 🔷 When to Use ARIMA

* Data has trend but NO seasonality
* Data becomes stationary after differencing
* Suitable for short-term forecasting

---

# 🔷 When NOT to Use ARIMA

* Strong seasonality → Use SARIMA
* Multiple variables → Use VAR
* Non-linear patterns → Use ML/DL models
