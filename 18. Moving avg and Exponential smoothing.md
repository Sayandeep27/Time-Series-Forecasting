# 📊 Moving Average & Exponential Smoothing (Time Series)

---

## 📌 Overview

This document explains two fundamental smoothing techniques used in **Time Series Analysis**:

* **Moving Average (MA)**
* **Exponential Smoothing (ES)**

Both methods help in:

* Reducing noise
* Identifying trends
* Preparing data for forecasting models

---

# 🔹 1. Moving Average (MA)

## 📌 Definition

**Moving Average** is a technique that calculates the average of a fixed number of past observations to smooth a time series.

👉 Each value gets **equal weight**

---

## 🧠 Intuition

"Take the last few values and compute their average"

---

## 📐 Formula

```
MA_t = (Y_t + Y_{t-1} + Y_{t-2} + ... + Y_{t-k+1}) / k
```

Where:

* `MA_t` = Moving Average at time t
* `Y_t` = Actual value at time t
* `k` = Window size

---

## 📊 Example Dataset

| Day | Sales |
| --- | ----- |
| 1   | 10    |
| 2   | 12    |
| 3   | 14    |
| 4   | 13    |
| 5   | 15    |

---

## 🔢 3-Day Moving Average Calculation

### Step-by-step

```
MA_3 = (10 + 12 + 14) / 3 = 12
MA_4 = (12 + 14 + 13) / 3 = 13
MA_5 = (14 + 13 + 15) / 3 = 14
```

---

## 📊 Result Table

| Day | Sales | 3-Day MA |
| --- | ----- | -------- |
| 1   | 10    | —        |
| 2   | 12    | —        |
| 3   | 14    | 12       |
| 4   | 13    | 13       |
| 5   | 15    | 14       |

---

## ✅ Key Characteristics

* Equal weight to all observations
* Simple and easy to implement
* Uses a **fixed window size**
* Slower to react to sudden changes

---

# 🔹 2. Exponential Smoothing (ES)

## 📌 Definition

**Exponential Smoothing** assigns **more weight to recent observations** and less weight to older data.

---

## 🧠 Intuition

"Recent data is more important than older data"

---

## 📐 Formula

```
S_t = αY_t + (1 - α)S_{t-1}
```

Where:

* `S_t` = Smoothed value at time t
* `Y_t` = Actual value at time t
* `α` = Smoothing factor (0 < α < 1)

---

## 📊 Example Dataset

| Day | Sales |
| --- | ----- |
| 1   | 10    |
| 2   | 12    |
| 3   | 14    |
| 4   | 13    |
| 5   | 15    |

---

## 🔢 Calculation (α = 0.5)

### Step 1: Initialize

```
S_1 = Y_1 = 10
```

---

### Step 2: Compute

```
S_2 = 0.5(12) + 0.5(10) = 11
S_3 = 0.5(14) + 0.5(11) = 12.5
S_4 = 0.5(13) + 0.5(12.5) = 12.75
S_5 = 0.5(15) + 0.5(12.75) = 13.875
```

---

## 📊 Result Table

| Day | Sales | ES (α=0.5) |
| --- | ----- | ---------- |
| 1   | 10    | 10         |
| 2   | 12    | 11         |
| 3   | 14    | 12.5       |
| 4   | 13    | 12.75      |
| 5   | 15    | 13.875     |

---

## ✅ Key Characteristics

* More weight to recent values
* Uses all past data (infinite memory)
* Controlled by α (alpha)
* Faster response to changes

---

# 🔥 Moving Average vs Exponential Smoothing

| Feature           | Moving Average  | Exponential Smoothing    |
| ----------------- | --------------- | ------------------------ |
| Weights           | Equal           | Higher for recent values |
| Memory            | Fixed window    | Infinite memory          |
| Responsiveness    | Slow            | Fast                     |
| Complexity        | Simple          | Moderate                 |
| Control Parameter | Window size (k) | Alpha (α)                |

---

# 🎯 Important Insights

* **Moving Average** is best when:

  * Data is stable
  * Noise reduction is needed

* **Exponential Smoothing** is best when:

  * Recent trends matter more
  * Quick adaptation is required

---

# 💻 Python Implementation

## 🔹 Moving Average

```python
import pandas as pd

sales = [10, 12, 14, 13, 15]
df = pd.DataFrame({'Sales': sales})

# 3-day moving average
df['MA_3'] = df['Sales'].rolling(window=3).mean()

print(df)
```

---

## 🔹 Exponential Smoothing

```python
import pandas as pd

sales = [10, 12, 14, 13, 15]
df = pd.DataFrame({'Sales': sales})

# Exponential smoothing
df['ES'] = df['Sales'].ewm(alpha=0.5).mean()

print(df)
```

---

# 📌 Summary

* Moving Average = Equal weighting, simple smoothing
* Exponential Smoothing = Weighted smoothing, more realistic
* ES is generally preferred in real-world forecasting
